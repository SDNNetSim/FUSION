Claude Code
===========

We recommend using `Claude Code <https://docs.anthropic.com/en/docs/claude-code/overview>`_ to explore,
understand, and contribute to FUSION. Claude Code is Anthropic's AI-powered CLI tool that
understands your codebase and can help you navigate complex projects like this simulator.

Why Use Claude Code?
--------------------

FUSION is a complex simulation framework with many interconnected components. Claude Code helps you:

- **Learn the codebase**: Ask questions instead of reading thousands of lines
- **Understand concepts**: Get explanations of optical networking terminology
- **Write code faster**: Generate implementations following FUSION's patterns
- **Debug issues**: Describe problems and identify root causes
- **Contribute effectively**: Understand standards before submitting PRs
