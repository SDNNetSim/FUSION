Path Agent
==============

The ``path_agent.py`` module defines the ``PathAgent`` class, used to assign and update path selections during
reinforcement learning simulations. It extends ``BaseAgent`` and supports algorithms such as Q-Learning,
bandits, and SB3-based DRL.
