==================
fusion.interfaces
==================

Abstract base classes and interfaces for plugin architecture.

.. contents:: Table of Contents
   :local:
   :depth: 2

Overview
========

The ``fusion.interfaces`` package defines abstract interfaces that enable
FUSION's plugin architecture for routing, spectrum assignment, SNR calculation,
and RL agents.
