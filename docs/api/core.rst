============
fusion.core
============

Core simulation engine and network management components.

.. contents:: Table of Contents
   :local:
   :depth: 2

Overview
========

The ``fusion.core`` package contains the fundamental components for network simulation,
including the SDN controller, routing logic, spectrum assignment, SNR measurements,
and simulation orchestration.
