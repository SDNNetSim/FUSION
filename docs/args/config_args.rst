Configuration Arguments
=======================

.. automodule:: arg_scripts.config_args
    :members:
    :undoc-members:

**Script Parameters**

* **sim_type:** Represents the simulator's Erlang calculation assumption.

  * **Valid options:** ``yue`` | ``arash``

  * **References:** `Complete this`

* **holding_time:** Mean holding time duration for the requests.

  * **Valid options:** Any floating point value

* **arrival_rate:** The inter-arrival rate of requests.

* **thread_erlangs:** To thread each traffic volume.

  * **Valid options:** (True, False)
