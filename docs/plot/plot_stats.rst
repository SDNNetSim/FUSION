Plot Statistics
=================

The Plot Statistics Script provides functions for visualizing simulation data, offering users the ability to
personalize plots by selecting specific simulations and styles. These utilities enhance the analysis process by
enabling users to customize plots according to their preferences and requirements, facilitating clear and insightful
visualization of simulation results.
