===============
Getting Started
===============

Welcome to FUSION! This section will help you get up and running with the simulator,
from installation to running your first simulation.

.. contents:: In This Section
   :local:
   :depth: 2

Overview
========

FUSION is a comprehensive simulation framework for Software Defined Elastic Optical Networks (SD-EONs).
Whether you're a student learning about optical networking, a researcher developing new algorithms,
or a professional evaluating network designs, FUSION provides the tools you need.

Prerequisites
=============

Before installing FUSION, ensure you have:

* **Python 3.11.x** installed on your system
* **Git** for cloning the repository
* Basic familiarity with command-line interfaces
* (Optional) Understanding of virtual environments in Python

Supported Platforms
===================

FUSION supports the following operating systems:

* **macOS** (Intel and Apple Silicon)
* **Ubuntu 20.04+**
* **Fedora 37+**
* **Windows 11**
